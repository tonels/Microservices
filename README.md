## aliCloud
集成了SpringCloud-Alibaba + Nacos + Sentinel，这种微服务架构，基于Nacos-config作为服务的注册和发现，基于Nacos-discovery作为分布式配置的统一管理。基于Sentinel作为服务监控，包括熔断和降级规则的配置。
